The Mission

Objective

A competition under ROBU (Robotics Club of BRAC University) challenging participants to create a fully functional website using ChatGPT-generated code and content only. Focus on creativity, design sense, and prompt crafting skills.

Eligibility

Open to all current HR & IT Department members. Individual competition (teams allowed). Participants can also join Project Showcase & Line Follower Race.

Rules & Guidelines

ChatGPT Only

Must generate all code and content using ChatGPT. Only minor manual edits allowed.

Allowed Tech

HTML, CSS, JS, PHP. No frameworks. VS Code + Live Server.

Duration

Total time: 1 Hour. Fast, efficient, and creative.

Disqualification

Plagiarism, prewritten code, or failure to submit prompt history leads to disqualification.

Evaluation Criteria

Creativity (40)

  • Parallax Effect (10)
  • Theme Consistency (10)
  • Animation & Icons (10)
  • Color & Outlook (10)

Functionality (30)

  • UI/UX (10)
  • Responsiveness (10)
  • Navigation (10)

AI Usage (20)

  • Prompting Style
  • Structure
  • Effective Usage

Bonus (20)

  • Backend Features
  • Login/Auth
  • Search/Sort
